## 这是什么
这是一个基于 [抖音](https://www.douyin.com) 和 [bilibili](https://www.bilibili.com) WEB API编写的 `Yunzai-Bot` 视频解析插件<br>
通过接口获取数据并渲染图片返回
### 特点
- 使用官方接口，不会出现类似 `服务商突然跑路` `被服务商拉黑` 的问题<br>
- 支持譬如 *评论* *番剧* 等等的解析功能
::: details 鸣谢
**业务站点**

- [www.douyin.com](https://www.douyin.com) & [www.bilibili.com](https://www.bilibili.com)
