# 推送功能
该功能为定时任务，默认每10分钟执行一次推送（可自定义推送间隔）
## 命令
|| 抖音       | B站      |
| --------------| -------------- | ---------------- |
|  |  `#设置抖音推送 + 抖音号`  | `#设置B站推送 + UID`|
|  例子|  `#设置抖音推送yuanshen_mihoyo`  | `#设置B站推送401742377`|

::: tip
在发送一次 `相同的命令` 即可取消在当前群的推送
:::


## 关于命令: `#kkk推送列表`

该命令需要获取推送后的缓存数据<br>
若刚刚配置推送用户，该博主/UP可能 `不会马上出现` 在推送列表中<br>
因为需要 `等待推送缓存刷新` 后才可获取到该博主/UP的相关信息
